# elections-dm - Elections Data Mining

Анализ электоральных данных.

Репозиторий содержит скрипты для выкачки результатов выборов с сайта избиркома РФ.

http://www.izbirkom.ru

Данные публичны.

Выборы состоялись 08 сентября 2019 г.
